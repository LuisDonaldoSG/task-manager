## Getting Started

install dependencies:

```bash
npm i
```
## Node version

to install dependencies you ned:

```bash
version 18.18.0 of nodejs at least
```
## run local server

run the development server:

```bash
npm run dev
```
## run test

to run unit test:

```bash
npm test
```

## build project

to build project run:

```bash
npm run build
```

## run builded project

to run builded version project:

```bash
npm run start
```
